# Opaline FM Mobile プライバシーポリシー

最終更新日: 2026年7月14日

Opaline FM Mobile は、iPhoneで4OP FM音源方式の音色を演奏・編集するための無料アプリです。本アプリは、ユーザーの個人情報を収集しません。

## 収集するデータ

本アプリは、ユーザーを識別する情報、利用状況、位置情報、連絡先、広告ID、診断情報、その他の個人データを収集しません。

## 広告・解析・トラッキング

本アプリには広告を表示しません。また、アクセス解析、行動解析、トラッキング、第三者広告SDKは使用していません。

## ファイル・音色データ

本アプリでは、音色データやバンクデータを読み書きできます。これらのデータは、ユーザーが明示的に読み込み・保存・書き出しを行う場合を除き、端末内で処理されます。本アプリがこれらのファイルを開発者または第三者のサーバーへ送信することはありません。

## MIDI・オーディオ

本アプリは、接続されたMIDIコントローラから演奏情報を受信し、端末上で音声を生成します。MIDI入力や音声出力に関する情報は、演奏と設定表示のために端末内で使用されます。本アプリがMIDIデータや音声データを収集・送信することはありません。

## マイク

本アプリは音声出力を目的としたアプリであり、マイク入力を必要としません。

## 第三者への提供

本アプリは、ユーザーの個人データを収集しないため、第三者へ提供する個人データはありません。
